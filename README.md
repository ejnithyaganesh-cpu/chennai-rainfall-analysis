# Chennai Rainfall Analysis

## Overview
This project analyzes rainfall patterns across Tamil Nadu districts using data analysis techniques. The goal is to identify monthly, seasonal, and district-level variations in rainfall.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
District-wise rainfall dataset for India (filtered for Tamil Nadu)

## Analysis Performed
- Data cleaning and preprocessing
- District-wise rainfall comparison
- Monthly rainfall trend analysis
- Seasonal rainfall pattern analysis
- Visualization using bar charts and histograms

## Key Insights
- Highest rainfall occurs during Oct–Dec (Northeast Monsoon)
- Rainfall varies significantly across districts
- Certain districts consistently receive higher rainfall
- Seasonal trends dominate rainfall distribution

## Files in Repository
- `chennai_rainfall_analysis.ipynb` → Main analysis notebook  
- `tamil_nadu_rainfall.csv` → dataset  

## How to Run
1. Download the notebook
2. Open in Google Colab or Jupyter Notebook
3. Run all cells
